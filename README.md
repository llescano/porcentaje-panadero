# Porcentaje Panadero

Una aplicación Flutter para panaderos que permite calcular porcentajes de ingredientes basados en el método del porcentaje panadero.

## Características

- Gestión de ingredientes con información nutricional
- Creación y edición de recetas
- Cálculo automático de porcentajes panaderos
- Calculadora para escalar recetas según cantidad deseada
- Base de datos local SQLite
- Interfaz intuitiva y fácil de usar

## Tecnologías Utilizadas

- Flutter
- Dart
- SQLite (sqflite)
- Material Design

## Instalación

1. Clona este repositorio
2. Ejecuta `flutter pub get` para instalar las dependencias
3. Ejecuta `flutter run` para iniciar la aplicación
